TextWidth
=========

Calculate text width use canvas measureText() method.


## Install
    npm install textwidth

## Example
```
var tw = require('textwidth');
// 使用默认字体及大小
var width = tw("Hello textwidth!!!")
// 设置字体,大小
var width = tw("Set the text font", "10px Arial")
```
